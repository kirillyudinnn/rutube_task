## Требования

- Python 3.9
- См. файл [requirements.txt](requirements.txt) для версий модулей

## Установка

1. Склонируйте репозиторий:

    ```bash
    git clone https://github.com/kirillyudinnn/rutube_task.git
    ```

2. Установите зависимости:

    ```bash
    pip install -r requirements.txt
    ```
